⚡ 3 Parallel Phases

12 agents across 3 phases. Phase 1: research, SVG, topics, and files run simultaneously. Phase 2: post-push config. Phase 3: all 4 distribution drafts.

🎯 SEO Name Scoring

5 candidates scored on search saturation, keyword match, memorability, and category signal — ranked before any other input is collected.

🔬 Competitor Research

Agent B mines the top 3 similar repos' READMEs: headline format, install placement, CTA position, value prop style. Your README is written from a benchmark.

📅 7-Day Calendar

Not a list of platforms — a timed plan. Reddit on Day 1, X/Twitter Day 2, awesome-list Day 3, HN on Day 5 only after reaching 10 stars.
